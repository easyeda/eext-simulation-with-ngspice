# 更改日志

## V1.2.1

- 支持接收 EDA 仿真事件中的 `netlist` 与 `probeNodes`，自动打开波形面板并导入当前仿真网表。
- EDA 仿真事件导入网表后会自动运行仿真；原理图菜单仅保留 `打开波形界面`，用于手动导入或粘贴自定义网表。
- 配置 `onStartupFinished` 激活事件，确保插件在 EDA 启动完成后自动注册仿真事件监听。
- 新增 EDA 探针默认显示逻辑：仿真完成后按探针节点名匹配已有曲线，并在曲线筛选弹窗中默认选中对应波形。
- 新增 `XAM` 电流探针处理：自动补充探针两端电压保存项，并按 `(V(nodeA) - V(nodeB)) / 1e-3` 合成电流曲线。
- 修复同一份 EDA 网表消息多次延迟广播导致仿真完成后波形被清空的问题。
- 优化曲线选择弹窗打开时机，等待运行状态和图表布局刷新后再打开，减少首次运行时弹窗闪烁或波形不显示的概率。
- 修复 AC 分析频率单位兼容问题：`.ac` 命令中的 `1M`、`10M`、`1MHz` 会在运行前转换为 ngspice 可识别的 `1Meg`、`10Meg`。
- 转换范围仅限 `.ac` 命令的起始频率和终止频率参数，不会修改元件值、模型参数或其他分析命令。
- 按扩展广场审核要求，非根目录 README 已改名为说明文档，避免详情页解析到错误 README。
- 根目录 README 增加功能图和功能演示 GIF，便于扩展详情页展示完整仿真流程。

## V1.2

- 优化大数据波形渲染：默认只渲染当前横轴窗口内的数据点，减少长波形整体绘制压力。
- 缩小时使用保峰值下采样，按数据桶保留首尾、最大值和最小值，降低尖峰被普通抽样弱化的概率。
- 缩放和平移后会重新计算当前窗口渲染点，放大到局部时展示更多原始数据细节。
- 保留 V1.1 打包产物，V1.2 以独立 `.eext` 包输出。
- 对齐扩展广场发布元数据，补充自定义 LOGO 与 NGspice Modified BSD 许可信息。

## V1.1

- 作者与发布方更新为 runlong。
- 优化插件描述，明确本地 NGspice WASM、网表导入、波形查看和交互分析能力。
- 优化波形交互：按鼠标位置缩放，绘图区缩放横轴，电压 / 电流轴区域缩放对应纵轴。
- 自适应视图遵守有效数据边界，并只按当前未隐藏曲线计算范围。
- 图例图标更新为短横线加圆点样式。

## V1.0

- 内置真实 NGspice 46 WASM 引擎，运行仿真时不再下载或启动本地原生服务。
- 支持导出当前原理图仿真网表，也支持导入和粘贴纯 NGspice 网表。
- 支持 transient、AC、DC 仿真结果解析为 ECharts 波形渲染。
- 增加曲线选择、图例开关、波形全屏查看、缩放和拖拽观察能力。
- 优化 WASM 加载方式，使用内嵌二进制传入 Emscripten，适配 EasyEDA iframe 对 `.wasm` 文件拉取的限制。
- 增加底部日志模块，展示运行状态、stdout / stderr 和失败原因。
