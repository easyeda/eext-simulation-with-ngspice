# 更改日志

## V1.2.2

- 新增当前波形数据导出功能：在波形工具栏中导出当前激活数据集为 CSV，包含 X 轴和该数据集全部曲线列。
- 新增运行日志导出功能：可将当前界面的运行日志导出为 `.log` 文件，便于问题定位和结果归档。
- 导出保存优先使用 EDA 宿主文件系统接口，独立预览环境自动回退到浏览器下载。
- 优化波形工具栏布局：徽标和按钮组分离，按钮保持单行显示，避免“放大”按钮被挤到第二行。
- 修复打包脚本在当前 TypeScript / Node 类型组合下 `JSZip.file` 不接受 `Buffer` 导致 `.eext` 打包失败的问题。

## V1.2.1

-  `XAM` 电流探针处理：自动补充探针两端电压保存项，并按 `(V(nodeA) - V(nodeB)) / 1e-3` 合成电流曲线。
- 修复同一份 EDA 网表消息多次延迟广播导致仿真完成后波形被清空的问题。
- 优化曲线选择弹窗打开时机，等待运行状态和图表布局刷新后再打开，减少首次运行时弹窗闪烁或波形不显示的概率。
- 新增数值线模式切换：`跟随` 模式随鼠标移动，`游标` 模式在画布上常驻，顶部红色三角手柄可拖动定位。
- 修复 AC 分析频率单位兼容问题：`.ac` 命令中的 `1M`、`10M`、`1MHz` 会在运行前转换为 ngspice 可识别的 `1Meg`、`10Meg`。
- 转换范围仅限 `.ac` 命令的起始频率和终止频率参数，不会修改元件值、模型参数或其他分析命令。
